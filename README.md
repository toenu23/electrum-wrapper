# electrum-wrapper
Minimal wrapper for Electrum Bitcoin API, supports TCP and TLS connections

### Installation
`npm install electron-wrapper`

### How to use
See [examples](https://github.com/toenu23/electrum-wrapper/blob/master/examples/example.js)
